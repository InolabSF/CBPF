## Filter Crime Data

Store crime data if crime description includes these words because they relate to cycling.

```ruby
'STREET'
'SIDEWALK'
'TRAFFIC'
'OBSTRUCTING'
'LOITERING'
'PEDDLING'
'PARK'
'VEHICLE'
'BICYCLE'
'AUTO'
'MOTORCYCLE'
'TRUCK'
'TAXI'
'DRIVER'
'LICENSE'
'ALCOHOL'
'DRUNK'
'SPEEDING'
'FIREARM'
'GUN'
'SHOOTING'
'DRUG'
'MISCHIEF'
'ATM'
'MANNER'
'DISTURB'
'INDECENT'
```
