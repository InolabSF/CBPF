# CrimeData Collection

<br />


## Method

1. Get crime data
2. Filter crime that relates bicycle, and save them to DB

<br />


## Filter crime

Save only crimes whose description includes these words

##### Strict vehicle crimes

- TRAFFIC
- BICYCLE
- STOLEN
- SPEEDING
- SLASHING

##### Other crimes

- ALCOHOL
- DRUNK
- FIREARM
- GUN
- SHOOTING
- DRUG
- STREET
- SIDEWALK
- LOITERING
- WEAPON
